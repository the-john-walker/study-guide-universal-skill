Universal Study Guide Skill

A Claude skill that builds a complete study plan as an interactive HTML file.

What it does


Asks a few questions about your goal, deadline, time available, and current level
Builds a full interactive HTML study plan based on your answers
Works for exam prep, school subjects, skill building, or project roadmaps


Triggers


"make me a study plan"
"help me prep for..."
"build me a schedule for..."
"how do I learn X"
"I have Y weeks to learn Z"


What the file includes


Week/unit selector with next/prev buttons
Progress bar
Time-boxed steps with built-in timers
Direct links to specific resources
Targets based on your actual starting level
Spaced repetition every other week
Short on time fallback for every week
Works fully offline


Session types


Exam/competition: review concept, worked example, practice problems, error log
School subject: read and take notes, practice questions, self-quiz, error log
Skill building: study technique, apply it, review your own work
Project roadmap: milestones with goals, deliverables, success criteria


Themes


Dark: Midnight, Obsidian, Hacker
Light: Clean, Paper, Academic


Setup

Drop SKILL.md into your Claude skills folder. No config needed.

Requirements

Claude with file creation enabled (Claude.ai, Claude Code, or Claude Desktop)
